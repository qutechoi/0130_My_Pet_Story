# 3×3 펫 스토리보드 그리드 생성 가이드

## 개요

반려동물 사진을 기반으로 고급 펫 포토그래피 스토리보드를 3×3 그리드 형식으로 생성하는 기능입니다. 9개의 다양한 앵글과 분위기의 프레임을 하나의 통합 이미지로 제공합니다.

## 특징

### 🎨 고급 펫 포토그래피
- **DSLR 퀄리티**: Canon EOS R5 / Sony A7R IV 스타일
- **일관된 캐릭터**: 모든 프레임에서 동일한 동물 외형 유지
- **따뜻한 시네마틱 무드**: 자연스럽고 전문적인 분위기

### 📐 3×3 그리드 구조
- **9개 프레임**: 각기 다른 앵글과 분위기
- **균등한 크기**: 모든 패널이 동일한 비율
- **4:5 비율**: 최적화된 출력 비율
- **깔끔한 그리드**: 테두리, 텍스트, 워터마크 없음

## 프레임 구성

### Frame 1: 정면 포트레이트 (히어로샷)
- 카메라를 정면으로 바라보는 클래식 스튜디오 포트레이트
- 차분하고 자신감 있는 눈 맞춤
- 중립적 스튜디오 배경

### Frame 2: 극접사 (매크로)
- 눈 또는 코에 초점
- 젖은 코의 질감이나 홍채 깊이 강조
- 선명한 디테일, 가장자리 소프트 포커스

### Frame 3: 환경샷
- 아늑하고 세련된 거실에서 편안하게 앉거나 누워있는 모습
- 창문에서 들어오는 부드러운 자연광
- 따뜻하고 편안한 분위기

### Frame 4: 인터랙션
- 사람의 손이 부드럽게 머리를 쓰다듬거나 악수하는 장면
- 행복하거나 신뢰하는 반응에 초점
- 미니멀한 구도

### Frame 5: 탑다운 뷰 (하이앵글)
- 카메라가 위에서 아래를 내려다보는 구도
- 동물이 올려다보는 귀여운 눈빛
- 대칭적이고 사랑스러운 시점

### Frame 6: 액션샷
- 간식을 잡거나, 카메라를 향해 달리거나, 점프하는 동작
- 귀가 펄럭이는 역동적인 포즈
- 빠른 셔터 스피드로 순간 포착

### Frame 7: 디테일샷
- 발바닥(젤리), 꼬리, 귀 질감 등 특정 부위에 초점
- 부드러움과 털 디테일을 강조하는 예술적 구도

### Frame 8: 프로필 실루엣
- 옆을 바라보는 동물
- 드라마틱한 림라이트 또는 무드 조명
- 에디토리얼하고 세련된 느낌

### Frame 9: 휴식/수면
- 부드러운 쿠션이나 러그 위에서 웅크리거나 기지개를 켠 모습
- 평화롭고 고요하며 편안한 분위기

## 사용 방법

1. **반려동물 사진 업로드** — 고품질의 반려동물 사진 선택
2. **스토리보드 생성** — "Done" 버튼 클릭 후 30-60초 대기
3. **결과 확인** — 9개 프레임이 통합된 그리드 이미지 확인
4. **개별 프레임 확인** — Frame Details에서 각 프레임별 이미지와 설명 확인
5. **다운로드** — 전체 그리드 또는 개별 프레임 이미지 다운로드

## 카메라 & 스타일

- **렌즈**: 포트레이트 85mm, 환경샷 35mm
- **조명**: 부드럽고 확산된 조명으로 털 질감 강조
- **색감**: 크림, 브라운, 따뜻한 그레이 등 자연스러운 팔레트
- **일관성**: 9개 프레임 전체에서 통일된 비주얼 언어

## 기술 사양

### API 설정
```javascript
{
  model: "gemini-3-pro-image-preview",
  generationConfig: {
    responseModalities: ["IMAGE", "TEXT"],
    imageConfig: { imageSize: "1K" }
  }
}
```

### 이미지 사양
- **해상도**: 1024x1024 (1K)
- **비율**: 4:5 (전체 그리드)
- **형식**: PNG
- **그리드 분할**: Canvas API로 9등분하여 개별 프레임 표시

## 최적의 결과를 위한 팁

### 좋은 원본 이미지
- 고해상도 (최소 1000x1000px)
- 동물이 명확하게 보이는 사진
- 좋은 조명 조건
- 품종, 털 무늬, 눈 색깔이 잘 드러나는 사진

### 피해야 할 이미지
- 저해상도 또는 흐릿한 사진
- 여러 동물이 함께 있는 사진
- 너무 어두운 조명
- 동물이 작게 나온 사진

## 제한사항

1. **생성 시간**: 30-60초 소요
2. **동시 생성**: 1개씩만 가능
3. **편집 불가**: 생성 후 수정 불가 (재생성만 가능)
4. **고정 그리드**: 3×3 형식 고정
